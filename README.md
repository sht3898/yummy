#  YUMMY 명세서

## 🤔목적

* 국내 맛집 여행 계획 플래너 서비스 재공
* 대분류(광역시/시/군)별 일정 서비스 제공
* 소분류(일정)별 시간 계획 서비스 제공
* 사용자 기반 추천 음식점 정보를 계획한 관광지/숙소별 거리 기준 제공

## 💡 주요 기능

* 일정 서비스 제공
* 일정별 시간 계획 서비스 제공
* 사용자 기반 추천 맛집 제공

## 📚데이터 활용

* SSAFY 음식점 데이터
* KaKao(MAP)
* 광역/지방 자치회 센터(시청, 군청) 데이터
* 공공API( 전국민박/펜션업소표준데이터 )
* 공공API( 전국관광지정보표준데이터 )

## 🤝 팀 소개

**삼성 청년 S/W 아카데미(SSAFY) 2기 대전 2반 공통 프로젝트 7팀**
